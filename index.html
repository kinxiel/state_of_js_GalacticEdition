State of JS: Galactic Edition

In my poor attempt to mimic the design ambiance of the State of JS 2018, in this alternative world, web technologies are represented by planets rotating around a main theme (Front-End, Back-End, Data, Flavor, Testing).

  • The size of the planet represents the total number of users
  • The period/speed of the planet represents the amount of interest in the technology
  • For perspective, the orbits represent the initial release of the technology (or at least my best estimate of it)

Insights

  • For Front-End Frameworks, we can see that there is a slight correlation between interest and newness. We can see that although React has a large user base and steady interest, Vue is gaining steam quite rapidly.
  • The situation in Back-End Frameworks is quite unique. Although, Express is already 8 years old, it still garners a lot of interest from developers.
  • For some reason most of the tech in the Data Layer section more or less started around the same time; fierce competition?
  • For newer developers aiming to become full stack developers, the size of the planets may serve as a useful guide. Bigger planets (more users) can indicate that the technology is mature and that resources are available. [React, Express, Redux, ES6, TypeScript, Jest] (don't know if that combination makes sense...)
  • For people who want to get ahead of the trend, you might want to look at the planets with the most momentum. [Vue, Next.js, GraphQL, TypeScript, Jest]
